## 📌 Project Overview

This project is a **Job Application Form** built using pure HTML. It is designed to collect all relevant information from a job applicant in a structured, user-friendly layout — covering contact details, position applied for, resume upload, education, skills, and availability preferences.

It serves as a foundational front-end project demonstrating the use of various **HTML form elements**.

---

## 🚀 Features

| Feature | Description |
|---|---|
| 👤 **Contact Information** | Collects first name, last name, email, and phone number |
| 💼 **Position Details** | Text input for the role being applied to |
| 📄 **Resume Upload** | File input for uploading a resume |
| 🌐 **Portfolio Link** | Text input for a portfolio website URL |
| 🏠 **Relocation Preference** | Radio buttons for Yes / No / Not Sure |
| 🎓 **Education Level** | Dropdown for Undergraduate, Postgraduate, PhD |
| 🛠️ **Skills Checklist** | Checkboxes for HTML, CSS, JavaScript, MERN |
| 🔢 **Experience** | Number input (0–10 years) |
| 🕐 **Availability** | Radio buttons for Full Time / Internship |

## ⚙️ Setup & Usage

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/ManwaniDiksha/job-application-form.git
cd job-application-form
```

### 2️⃣ Open in Browser

Simply open the file directly in any browser:
```bash
# Just double-click Project1.html
# OR open via terminal:
start Project1.html        # Windows
open Project1.html         # macOS
xdg-open Project1.html     # Linux
```

No server or dependencies required — it's pure HTML!

## 🎯 Learning Outcome

This project demonstrates:

- Structuring forms using semantic HTML
- Using a variety of input types: `text`, `tel`, `email`, `file`, `number`, `radio`, `checkbox`, `select`
- Grouping related fields using `<fieldset>` and `<legend>`
- Building accessible forms with proper `<label>` and `for` attributes
